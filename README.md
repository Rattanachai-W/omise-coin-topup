# Omise Coin Top-up System

ระบบเติมเงินและชำระเงินออนไลน์ผ่าน **Omise Payment Gateway**  
รองรับการชำระเงินด้วย **PromptPay QR Code** และ **บัตรเครดิต / เดบิต**  
เมื่อชำระเงินสำเร็จ ระบบจะเพิ่ม **เหรียญ** เข้าไปยังบัญชีผู้ใช้โดยอัตโนมัติ

---

## Features

- เลือกจำนวนเงินสำหรับเติม
- รองรับ PromptPay QR Payment
- รองรับบัตรเครดิต / เดบิต
- สร้าง QR Code สำหรับชำระเงิน
- ระบบนับเวลาหมดอายุของ QR
- เพิ่มเหรียญเข้าระบบอัตโนมัติหลังชำระเงินสำเร็จ
